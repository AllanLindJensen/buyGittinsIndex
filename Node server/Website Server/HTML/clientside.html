The index is the fair price for a trial in units where an average failure is worth nought, and an average succes is worth unity.



more info on the Gittins Indices

In the computation it is assumed that a service is being tried, the value of which is a random variable with an unknown distribution in the interval [0,1]. Each time the service is tried, the value of the result is either above or below the 0.5. This gives new knowledge about the distribution of the output. If a player has a very long horizon, indicated by a discount factor close to 100 per cent, it may be worth trying a new service, expecting to find a better one in due time in stead of staying with a rather good one. This balance depending upon an infinite future is found by computation of the Gittins Index.
